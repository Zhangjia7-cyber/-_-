# 🔥 元素人格测试仪

一个基于 Streamlit 的趣味人格测试应用，通过问答方式判断用户的元素人格类型（火、水、土、风）。

## ✨ 功能特性

### 🎮 核心功能
- **元素人格测试**：5道趣味问答，判断您的元素人格类型
- **结果分析**：详细展示人格类型描述、优点、缺点和生活建议
- **动态形象**：可互动的元素动态形象，支持悬停放大和点击反馈
- **视觉效果**：渐变背景、毛玻璃卡片、发光标题

### 🎨 视觉设计
- 紫色渐变背景
- 毛玻璃效果问题卡片
- 趣味 emoji 装饰
- 选项按钮悬停放大变色
- 平滑进度条
- 响应式布局

### 💎 元素类型
| 元素 | 特点 | 优点 | 缺点 |
|------|------|------|------|
| 🔥 火元素 | 热情勇敢 | 热情、勇敢、充满活力 | 急躁、冲动、缺乏耐心 |
| 💧 水元素 | 温柔感性 | 温柔、感性、富有同理心 | 情绪易波动、优柔寡断 |
| 🌍 土元素 | 踏实稳重 | 踏实、稳重、注重实际 | 固执、保守、缺乏灵活性 |
| 💨 风元素 | 聪明好奇 | 聪明、好奇、善于思考 | 执行力弱、容易分心 |

## 📖 使用说明

1. **启动应用**：运行 `streamlit run project1.py`
2. **开始测试**：点击"开始测试"按钮
3. **回答问题**：选择符合您的选项（共5题）
4. **查看结果**：测试完成后显示您的元素人格类型
5. **重新测试**：点击"重新测试"按钮再次测试

## ❓ 常见问题

**Q: 测试结果准确吗？**

A: 这是一个趣味测试，仅供娱乐和自我探索。结果基于您的选择计算，平局时会显示混合元素。

**Q: 如何重新测试？**

A: 在结果页面点击"重新测试"按钮即可。

**Q: 可以分享测试结果吗？**

A: 可以截图分享您的元素人格类型结果。
